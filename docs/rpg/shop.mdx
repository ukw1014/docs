我們透過商店系統讓這個遊戲的經濟造成流通！快來開始加入販賣物品的行列

## 查看商店

使用 `&shop list` 來查看自己商店裡的物品，在後面加上別人的ID可查看對方的商店

![商店](/img/docs/rpg/shop/shop.png)

### 上架物品

<Adsense/>

恭喜你在進行了許多次 [工作](/docs/rpg/jobs/) 之後，賺到了許多的物資，該開始來販賣給其他人賺取收益了！

:::info 我想先通知你
為了避免這個遊戲的經濟崩盤，市場設有最低的上架價錢限制，這個最低價格和 `&sell` 直接賣給系統的價格是有關連性的，當然我們也希望促進各位去進行交易，並避免直接賣給系統，如果要進行出售可以查看 [出售給系統](#出售給系統)。
:::

使用 `&shop add 物品 數量 單一品項價格` 來進行上架，數量可以使用 `all` 來替代，將會使用背包內全部的數量，上架價格必須要高於最低的價格限制，

![商店 上架](/img/docs/rpg/shop/shop-add.png)

如果要在現有上架的物品再增加數量一樣可以使用 `&shop add` 物品 要增加的數量 價格` 放更多物品上去，然而架上最多可同時有`10`項不同的物品販售。
為了避免上架後，賣方利用訊息的時間差隨意竄改價格造成買方的無端的購買到不合理價格的商品，因此每次上架後有`15`秒的時間不能新增商品或者更動架上的價格。

### 移除物品

輸入 `&shop remove 物品` 將商店內的物品下架，下架後的數量將會全數歸還到背包裡。

## 出售給系統

使用 `&sell` 指令來出售物品給系統，出售之後所獲得的錢將會是該物品最低價格的 90% 價錢。
